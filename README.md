# DroneClassifierAndroid
This application can be used to load various TensorFlowLite ML Models to recognise drone sounds.

## How to use
1. Open the project in Android Studio.
2. Do not upgrade the Android SDK version if asked to do so.
3. Sync gradle and Build the project.
4. Connect your Android Phone to your PC, ensure Developer and Debug modes are enabled on the Phone.
5. Run the application with your phone selected as the target.
6. Confirm when asked for access to your Microphone.
7. Select the .tflite model you would like to run.
8. Click Start.

The Model output will be written to the centre button as text.
